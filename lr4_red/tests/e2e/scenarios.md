# E2E сценарии ЛР4 (текстовое описание)

1. Регистрация и вход:
   1. Пользователь отправляет `/auth/register`.
   2. Пользователь выполняет `/auth/login`.
   3. Ожидается HTTP `201` и затем HTTP `200`.

2. Создание брони:
   1. Пользователь авторизуется.
   2. Пользователь создает бронь через `/bookings`.
   3. Ожидается HTTP `201` и рассчитанное поле `total_cost`.

3. Оплата брони:
   1. Пользователь создает бронь.
   2. Пользователь вызывает `/bookings/{id}/pay`.
   3. Ожидается HTTP `200` и `paid=true`.

4. Ошибка валидации интервала:
   1. Пользователь отправляет бронь с `end_time <= start_time`.
   2. Ожидается ошибка `400`.

5. Стратегия турнира:
   1. Пользователь-участник турнира создает бронь.
   2. Ожидается `total_cost=0`.
