# Архитектура ЛР4

Проект реализует слоистую архитектуру:

- `src/api` - входные точки FastAPI, валидация запросов и маршрутизация.
- `src/services` - бизнес-логика и паттерн `Strategy` для расчета цены.
- `src/repositories` - слой доступа к данным (скелет, без реализации).
- `src/db` - ORM-модели и сессия SQLAlchemy AsyncIO.
- `src/schemas` - контракты Request/Response на Pydantic v2.

На этапе ЛР4 методы сервисов и репозиториев намеренно не реализованы.
