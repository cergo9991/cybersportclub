# E2E сценарии ЛР5 по DFD

1. Процесс 1 (Регистрация игроков):
   1. Игрок проходит регистрацию.
   2. Игрок входит в систему.

2. Процесс 2 (Создание турниров):
   1. Администратор входит в систему.
   2. Администратор создает турнир.

3. Процесс 3 (Распределение компьютерных залов):
   1. Администратор выбирает турнир.
   2. Администратор назначает зал/ПК для турнира.

4. Процесс 4 (Отслеживание заявок):
   1. Игрок подает заявку на участие.
   2. Администратор просматривает pending-заявки.
   3. Администратор одобряет или отклоняет заявку.

5. Процесс 5 (Ведение архива событий):
   1. Администратор добавляет результат турнира.
   2. Событие отображается в архивной ленте.
